# Калькулятор на WPF

Простой калькулятор с базовыми операциями, написанный на C# и WPF.

## Функции

- Сложение, вычитание, умножение, деление
- Кнопки `CE` (очистка текущего ввода), `C` (полный сброс), `←` (удаление последнего символа)
- Защита от деления на ноль

1. Открыть решение в Visual Studio.

## Технологии

- C# .NET 6/8
- WPF (XAML)
